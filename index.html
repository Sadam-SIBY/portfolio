Projets Les différents projets sur lesquels j'ai travaillé :

Fresh Shop

Lien dépôt Github
Boutique en ligne spécialisée dans la vente de sacs à dos pour ordinateurs portables, développée avec le framework Symfony. Bien que certaines fonctionnalités avancées puissent manquer, le principal but de ce projet était de me perfectionner dans l'utilisation de Symfony.
Backend : Symfony (PHP) Base de données : MySQL
Frontend : HTML, CSS, Twig, JavaScript API de paiement : Stripe

Juridiq

Plateforme de recherche juridique permettant aux juristes de retrouver facilement n'importe quel article de loi et décision de justice via un moteur de recherche par mot-clé et par filtres.
Backend : Symfony (PHP) Base de données : MySQL
Frontend : HTML, CSS, Twig, Bootstrap, JavaScript

O'Boulot

Lien dépôt Github
O'Boulot est un projet éducatif permettant aux enseignants de proposer des exercices et corrigés à leurs élèves. Cette application a été conçue et réalisée en équipe dans le cadre de ma formation à l'école O'Clock.
Backend : Symfony (PHP) Base de données : MySQL
Frontend : HTML, CSS, React.js

AlloFlix

Lien dépôt Github
Site répertoriant les nouveautés en termes de films et séries. Les films et séries sont classés par genre, permettant à l'utilisateur d'ajouter ceux qui l'intéressent à sa liste de favoris, de leur attribuer une note et de publier des critiques. Le site a été développé avec le framework Symfony.
Backend : Symfony (PHP) Base de données : MySQL
Frontend : HTML, CSS, Twig, Bootstrap, JavaScript

Trip Odvisor

Projet de conseils touristiques visant à aider les gens à organiser leurs voyages en fournissant des avis, des guides et des bons plans sur les destinations les plus recherchées, ainsi que sur les restaurants les plus fins et les hôtels les plus luxueux, etc. Ce projet a été réalisé dans le cadre de ma formation chez O'Clock.
Frontend : HTML, CSS, Bootstrap, JavaScript

O'Shop

Site e-commerce, en l'occurrence une boutique en ligne de chaussures. Le développement du site a suivi la méthodologie agile Scrum, basée sur un développement itératif par Sprints.
Frontend : HTML, CSS, Bootstrap, JavaScript
